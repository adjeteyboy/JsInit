1. What are variables

Variables are a way to temporarily save something in javascript

In javascript we use the 'var' keyword to define a variable followed by the NAME then an equal sign then the value.
Like so var name = 'Eric'

You will notice that i put the name Eric in inverted commas. This is to tell javascript that our variable is plain text. Plain texts are called STRINGS. I strongly advise that you use the inspector in your browser to inspect this code.

You can get to the DEVELOPER TOOLS by right clicking on the page and then selecting INSPECT.
